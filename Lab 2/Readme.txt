Model - Html filen
Det er denne filen som er modellen og holder på dataen vi har.

View - CSS filen
Denne filen bestemmer hvordan dataen vår skal vises.

Controller - Javascript filen
Denne filen kontrollerer hva som skal vises av informasjonen i HTML filen.
Hele poenget med disse er å minske repetisjon av data og gjøre det mer oversiktelig.
